Real-Time-OS Scheduler

This project is a basic simulation of a Real-Time Operating System (RTOS) scheduler using Rate Monotonic Scheduling (RMS) in C++. The scheduler manages a set of periodic tasks, each with a defined execution time and period. The RMS algorithm is used to determine the order in which tasks are executed based on their periods. Understanding of basic operating system principles, such as task scheduling and periodic task management helps in implementation of scheduling for real-time tasks, ensuring tasks are executed within their defined periods.

Wireless Modem Simulation

The Wireless Modem Simulation Project is a comprehensive simulation tool designed to model and demonstrate the behavior of different wireless modems, including CDMA, GSM, UMTS, and LTE. This project simulates real-time data transmission and reception using these modem technologies, providing insights into their functionalities and data handling mechanisms. It incorporates real-time data generation to mimic actual data transmission scenarios. The simulation includes basic encoding and decoding to illustrate data handling processes.
